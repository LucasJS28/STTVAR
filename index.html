STTVAR

Transcripción Colaborativa y Análisis con IA en Tiempo Real

La herramienta definitiva que convierte la voz en conocimiento accesible y colaborativo. Una plataforma interactiva y multiusuario, potenciada por IA y disponible desde cualquier lugar del mundo.

Descargar en GitHub Ver Características

¿Qué es STTVAR?

STTVAR opera bajo un modelo Servidor-Cliente. La aplicación de escritorio actúa como un potente Servidor que transcribe, graba y sirve los datos en tiempo real. La interfaz web (Cliente) permite a colaboradores conectarse vía código QR para ver la transcripción en vivo, chatear, tomar apuntes y analizar el texto, todo de forma simultánea.

Características Estelares

Core & Desktop Server (PyQt5)

Transcripción Instantánea

Convierte voz a texto eligiendo entre español o inglés, adaptando el modelo Vosk dinámicamente.

Grabación Concurrente

Captura y guarda el audio original (.wav) de alta calidad mientras la transcripción se ejecuta.

Gestión de Transcripciones

Interfaz para explorar, editar, consultar la IA, traducir y exportar en formatos PDF, Word y Markdown.

Interfaz Web & Colaboración Global

Traducción Individual

Cada usuario web selecciona su idioma. La aplicación traduce la transcripción en vivo para cada cliente de forma independiente.

Chat y Apuntes Colaborativos

Chat global en tiempo real y panel de apuntes que pueden compartirse con el grupo como un bloque descargable.

Asistente de IA Integrado

Los usuarios pueden realizar preguntas sobre el contexto de la transcripción, gracias a la integración con Ollama.

Acceso Global con un Clic

La integración con Ngrok genera una URL pública para compartir la sesión en internet, sin configuración manual.

Acceso con Código QR

Un QR generado en el escritorio permite unirse a la sesión fácilmente desde cualquier dispositivo en la red.

Streaming SSE

La transcripción se "empuja" a todos los clientes web en tiempo real sin recargas (Server-Sent Events).

Evolución del Proyecto

Fase 1: El Núcleo

Creación del sistema base de transcripción y el servidor Flask para enviar el texto en tiempo real.

Fase 2: Mejoras de UI/UX

Implementación de contadores, modo Zen, y la capacidad de tener un panel de apuntes personal.

Fase 3: Colaboración

Añadido el chat global para comunicación entre usuarios y la funcionalidad de compartir apuntes.

Fase 4: Inteligencia

Integración de un modelo de IA (Ollama) para permitir consultas sobre el texto y acciones rápidas.

Fase 5: Accesibilidad Global

Automatización de Ngrok para permitir compartir la sesión públicamente con un solo clic.

Pila Tecnológica

Backend y Lógica

Python: El lenguaje principal.
PyQt5: Para la interfaz de escritorio.
Flask: Como servidor web para la comunicación.
PyNgrok: Para la creación de túneles públicos.

Frontend

HTML5, CSS3, JavaScript: La base de la interfaz web. Sin frameworks, para un rendimiento máximo y control total del streaming de datos.

IA y Voz

Vosk: Modelo de reconocimiento de voz offline.
Ollama & Mistral: Para el procesamiento de lenguaje natural.
Argos Translate: Para las traducciones en tiempo real.

Guía de Inicio Rápido

1

Descarga los Modelos Vosk

Visita Vosk Models, descarga los modelos para español e inglés, descomprímelos y colócalos en la raíz del proyecto.

2

Instala las Dependencias

Asegúrate de tener Python 3.9+ instalado y ejecuta el siguiente comando en tu terminal:

pip install -r requirements.txt
3

Configura Ollama

Instala Ollama desde ollama.com y descarga el modelo de IA ejecutando:

ollama pull mistral:7b-instruct-q4_K_M
4

Ejecuta el Proyecto

Inicia la aplicación. ¡Listo para transcribir y colaborar!

python main.py
