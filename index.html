¿Qué es STTVAR?

STTVAR es un ecosistema completo que transforma cualquier ordenador en un potente servidor de transcripción, permitiendo a múltiples usuarios colaborar en tiempo real desde una sencilla interfaz web.

¿Cómo Funciona?

La magia de STTVAR reside en su arquitectura Servidor-Cliente. La aplicación de escritorio hace todo el trabajo pesado, y la interfaz web sirve como una ventana universal para la colaboración.

Evolución del Proyecto

Fase 1: El Núcleo

Creación del sistema base de transcripción y el servidor Flask para enviar el texto en tiempo real.

Fase 2: Mejoras de UI/UX

Implementación de contadores, modo Zen, y la capacidad de tener un panel de apuntes personal.

Fase 3: Colaboración

Añadido el chat global para comunicación entre usuarios y la funcionalidad de compartir apuntes.

Fase 4: Inteligencia

Integración de un modelo de IA (Ollama) para permitir consultas sobre el texto y acciones rápidas.

Fase 5: Accesibilidad Global

Automatización de Ngrok para permitir compartir la sesión públicamente con un solo clic.

Guía de Instalación

Poner en marcha STTVAR es un proceso sencillo. Sigue estos cuatro pasos para tener tu servidor de transcripción funcionando en minutos.

1

Prepara el Reconocimiento de Voz

Descarga el modelo en español desde Vosk Models, descomprímelo y coloca la carpeta en la raíz del proyecto.

2

Instala las Dependencias

Abre una terminal en la carpeta del proyecto y ejecuta el siguiente comando para instalar todas las librerías necesarias.

pip install -r requirements.txt
3

Activa la Inteligencia Artificial

Instala Ollama desde su página oficial. Luego, ejecuta este comando para descargar el modelo de IA que usará la aplicación.

ollama pull mistral:7b-instruct-q4_K_M
4

¡Lanzamiento!

Con todo configurado, simplemente ejecuta el archivo principal para iniciar la aplicación. ¡Listo para transcribir y colaborar!

python main.py
